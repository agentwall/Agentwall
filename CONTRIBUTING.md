# Contributing

Thanks for your interest in AgentWall.

## How to contribute

- **Issues** — Bug reports, feature ideas, and questions: [open an issue](https://github.com/agentwall/agentwall/issues).
- **Pull requests** — Fork the repo, branch from `main`, and open a PR with a clear description of the change.

## Development

- Node.js **>= 22**
- `npm install` then `npm run build`
- `npm run dev` to run the CLI from source

Please keep changes focused and match existing style where practical.
